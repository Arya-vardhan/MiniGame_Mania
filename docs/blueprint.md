# **App Name**: MiniGame Mania

## Core Features:

- Game Selection: Allow users to select from a list of available mini-games.
- Ludo Game: Implement a playable Ludo game.
- Sentence Puzzles: Offer sentence puzzle games, that could involve filling in the blanks or unscrambling words. Leverage a tool to select an age-appropriate puzzle for the user to play.
- Tic Tac Toe: Implement a playable Tic Tac Toe game.
- Truth or Dare: Presents a game of Truth or Dare.
- Leaderboard: Keep track of scores and display a leaderboard for each game.
- User Profiles: Allow users to create profiles to save progress and preferences (saved in local storage).

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to create a playful and engaging atmosphere.
- Background color: A very light, desaturated blue (#E5F5F9), creating a clean and calming backdrop.
- Accent color: A bright orange (#FF8C00), adding a touch of excitement and calling attention to key interactive elements.
- Font: 'Inter', a grotesque-style sans-serif. This font will be used for the entire app.
- Use clear, consistent, and playful icons for game selection and in-game actions.
- Design a responsive layout that adapts to different screen sizes, making the games accessible on various devices.
- Incorporate subtle animations and transitions to enhance user engagement and provide feedback for interactions.
